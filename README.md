# Splitty - Expense splitting web app (Boilerplate)

This repository contains a starter web app for tracking shared payments between friends.
Stack:

-   Go (Fiber v3)
-   Postgres
-   UnoCSS (tailwind reset)
-   sqlc config included (you can run `sqlc generate` after installing sqlc)
-   templ (templating engine) + HTMX + AlpineJS
-   `air` for live reload in development
